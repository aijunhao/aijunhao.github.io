Python学习笔记——函数式编程

函数式编程的特点：

  1. 把计算视为函数而非指令
  2. 纯函数式编程：不需要变量，没有副作用，测试简单
  3. 支持高阶函数，代码简洁

Python支持的函数式编程：

  1. 不是纯函数式编程：允许有变量
  2. 支持高阶函数：函数也可以作为变量传入
  3. 支持闭包：有了闭包就能返回函数
  4. 有限度地支持匿名函数

高阶函数

变量可以指向函数，函数的参数能接收变量，一个函数可以接收另一个函数作为参数，这种函数就称之为高阶函数。

返回函数

高阶函数除了可以接受函数作为参数外，还可以把函数作为结果值返回。

闭包

闭包三要素：

  1. 必须有一个内嵌函数(函数里定义的函数）——这对应函数之间的嵌套
  2. 内嵌函数必须引用一个定义在闭合范围内(外部函数里)的变量——内部函数引用外部变量
  3. 外部函数必须返回内嵌函数——必须返回那个内部函数

如返回函数中的例子就是闭包。

匿名函数：lambda表达式

不显地定义函数，在以前博客中常使用。

谨慎使用lambda函数

  1. lambda函数主要用作一些特定函数或方法的参数
  2. lambda函数有一些固定使用方法
  3. 一般情况下，建议使用def定义的普通函数

匿名函数有个限制，就是只能有一个表达式，不用写return，返回值就是该表达式的结果。

假设我们要增强某个函数的功能，但又不希望修改now()函数的定义，这种在代码运行期间动态增加功能的方式，称之为“装饰器”（Decorator）。

装饰器其实就是一个闭包。
