给定一个list或tuple，我们通过for循环来遍历这个list或tuple，这种遍历我们称为迭代（Iteration）。

在Python中，迭代是通过for … in来完成的，Python的for循环抽象程度高于C的for循环，可以作用于任何可迭代对象上。
