Hello, Hexo

Hexo 使用介绍

创建博客

在命令行中找到hexo安装路径，使用命令：hexo new post “博客名”，创建博客，博客路径为”安装路径/source/_posts/博客名.md”

生成及部署博客

编辑好文章，使用命令：hexo d -g，即可生成及部署，此时可以访问个人网站进行访问了：https://aijunhao.github.io/。
