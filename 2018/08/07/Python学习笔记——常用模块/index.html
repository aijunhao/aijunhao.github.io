Python学习笔记——常用模块

Python模块篇

模块导入

import语句

在文件顶部位置导入模块，然后使用。

from … import * 语句

不适用于大文件，容易重名冲突。

dir()函数

返回模块中所有方法。

random模块

random模块主要用于生成随机数，由梅森旋转算法生成的伪随机数。

函数 描述
seed(N) 初始化给定的随机数种子，默认为当前系统时间
random() 生成一个[0,1)之间的随机小数，与随机数种子有关
randint(a,b) 生成一个[a,b]之间的整数
randrange(m,n[,k]) 生成一个[m,n)之间以k为步长的随机整数
getrandbits(k) 生成一个k比特长的随机整数
uniform(a,b) 生成一个[a,b]之间的随机小数
choice(list) 从序列中随机选择一个元素
shuffle(list) 将序列seq中元素随机排列,返回打乱后的序列

os模块

os模块，系统操作模块。

函数名 描述
getcwd() 获取当前文件路径
rename(path) 重命名文件
chdir(path) 改变当前工作目录
mkdir(path) 创建单层目录
makedirs(path) 创建多层目录
rmdir(path) 删除空目录
remove(path) 删除指定文件
system(command) 运行系统shell命令
os.curdir 指代当前目录
os.pardir 指代上一级目录
os.sep 输出操作系统特定的路径分隔符(win下为’\\‘，Linux下为’/‘)
os.linesep 当前平台的终止符(win下为’\r\n’，Linux下为’\n’)
os.name 指代当前使用的操作系统
