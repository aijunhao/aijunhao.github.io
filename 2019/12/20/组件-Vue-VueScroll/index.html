组件-Vue-Vuescroll富文本组件

前言

Vuescroll 是一款基于 Vue.js 自定义滚动条的插件，它有两种模式：

  • native： 适用于 PC 端， 支持基本的自定义滚动条。
  • slide： 适用于移动端， 支持下拉-加载，上拉刷新，轮播等。

安装

在 vue-cli 3 中的配置

配置项

配置项主要是 vuescroll 滚动条模式 ，scrollPanel 滚动面板 ，rail 滚动条轨道 ，bar 滚动条滑块 ，scrollButton 滚动按钮（滚动条两侧的三角形按钮） 。

vuescroll

滚动条模式，

mode *

  • 类型：String`
  • 可选值： native | slide
  • 默认值： native

选择一个模式， native 或者 slide 。native 适合 pc 端，与原生滚动条相似；slide 适合移动端，支持下拉-加载，上拉刷新，轮播等。

通用

sizeStrategy

  • 类型：number|percent

  • 默认值：percent

如果父容器不是固定高度，请设置为 number , 否则保持默认的percent即可。

detectResize

  • 类型：boolean

  • 默认值：true

是否检测内容尺寸发生变化。

mode: ‘native’ 模式

wheelScrollDuration
  • 类型: number
  • 默认值: 0

滚轮滚动一段距离所需时间。

wheelDirectionReverse
  • 类型: boolean
  • 默认值: false

是否使滚轮方向相反。

mode: ‘slide’ 模式

以下所有内容只有在 slide 模式下可用。

pullRefresh *
  • 类型: Object
  • 默认值:

下拉刷新。

pushLoad *
  • 类型: Object
  • 默认值:

上拉加载。

paging
  • 类型：boolean
  • 默认值：false

是否启用分页模式。

scrollPanel

scrollPanel 内容的包装. 我们通过改变 scrollPanel 的 scrollLeft 和 scrollTop 来使内容移动。

initialScrollY / initialScrollX

  • 类型： number | string | false
  • 默认值： false

组件 mounted 之后自动滚动的距离。 例如 100 or 10%

scrollingX / scrollingY *

  • 类型：boolean
  • 默认值：true`

是否启用 x 或者 y 方向上的滚动。

speed *

  • 类型：number
  • 默认值：300
  • 单位：毫秒

完成一次滚动的时间。

easing

  • 类型: string | undefined
  • 默认值: undefined

滚动动画。

部分动画缓动函数如下：http://www.xuanfengge.com/easeing/easeing/

  • easeInQuad
  • easeOutQuad
  • easeInOutQuad
  • easeInCubic
  • easeOutCubic
  • easeInOutCubic
  • easeInQuart
  • easeOutQuart
  • easeInOutQuart
  • easeInQuint
  • easeOutQuint
  • easeInOutQuint

verticalNativeBarPos

  • 类型: String
  • 可选值：right|left
  • 默认值: right

原生滚动条的位置。

rail

滚动条轨道。

background *

  • 类型: string
  • 默认值: #a5d6a7

轨道的背景色。

size *

  • 类型: string

  • 默认值: 6px

轨道的厚度。

opacity *

  • 类型: number
  • 默认值: 0

轨道的透明度。默认是 0，即完全透明不可见。

specifyBorderRadius

  • 类型: false | string
  • 默认值: 20px

是否指定轨道的 border-radius 属性，如果不那么将会自动设置。

gutterOfEnds

  • 类型: string
  • 默认值: 2px

垂直滚动条距离顶部/底部边框的距离，或水平滚动条距离左右边框的距离。

gutterOfSide

  • 类型: string
  • 默认值: 2px

垂直滚动条距离右侧边框的距离，或水平滚动条距离底部的距离。

keepShow

  • 类型: boolean
  • 默认值: false

是否即使 bar 不存在的情况下也保持显示。

如果当设置 scrollPanel.scrollingY: false ，即不开启水平滚动条，此时水平方向不滚动，此时原则上没有滚动条显示效果。如果设置 keepShow 为 true ，那么水平滚动条会显示出来。

border *

  • 类型: string
  • 默认值: none

轨道的边框。用法如同原生 css 的 border ，需要设置宽度/边框类型/颜色。

bar

滚动条滑块。

onlyShowBarOnScroll

  • 类型: boolean

  • 默认值: true

是否只在滚动时显示滑块。false 时只要鼠标移入滚动容器就会浮现滑块。

showDelay

  • 类型: number
  • 默认值: 500
  • 单位：毫秒

多久后隐藏滑块。

background

  • 类型: string
  • 默认值: #4caf50

滚动条背景色。

keepShow

  • 类型: boolean
  • 默认值: false

滚动条是否保持显示。设置为 true 时，onlyShowBarOnScroll 和 showDelay 将失效。

opacity

  • 类型: number
  • 默认值: 1

滚动条透明度。

specifyBorderRadius

  • 类型: false|string
  • 默认值: false

是否指定滚动条的 border-radius， 如果不那么和轨道的保持一致。

minSize

  • 类型: number
  • 默认值: 0

为滑块设置一个相对于轨道的最小尺寸, 从 0 到 1.0。 如 0.3, 代表 30%。

size

  • 类型: string
  • 默认值: 6px

滑块的厚度。

disable

  • 类型: boolean
  • 默认值: false

是否禁用滚动条。

scrollButton

轨道两端的三角形滚动按钮。

enable

  • 类型: blloean
  • 默认值: false

是否启用 scrollButton 。

background

  • 类型: string
  • 默认值: rgb(3, 185, 118)

背景色.

opacity

  • 类型: number
  • 默认值: 1

透明度。

step

  • 类型: number
  • 默认值: 180

每次当你点击 scrollButton 所滚动的距离。

mousedownStep

  • 类型: number
  • 默认值: 30

长按 scrollButton 时每次滚动的距离。

Api

通用 Api

scrollTo(position [, speed, easing])

  • position (Object)
    • x (string | number)
    • y (string | number)
  • speed (number)
  • easing (string)

滚动滚动到某一个位置，需要指定具体的 x 或 y。是一个定位的操作。

scrollBy(position [, speed, easing])

  • position (Object)
    • dx (string | number)
    • dy (string | number)
  • speed (number)
  • easing (string)

以当前位置为起点，滚动一段距离，dx 或 dy 。是一个定量的操作。

getCurrentviewDom()

获取你能看到的 vuescroll 的直接子元素.

scrollIntoView(selector[ , speed])

  • selector (string)
  • speed (number)

当前窗口滚动至 vuescroll 的一个直接子元素。选择器可以是任意选择器，id选择器 ，类选择器，ref 等。

getScrollProcess

获取当前的滚动进程，范围是[0,1]。返回值是一个对象 { v: 0, h: 0}，一个水平一个垂直。

Slide 模式的 Api

goToPage(page[, animate])

  • page (Object)
    • x (number)
    • y (number)
  • animate (boolean)

getCurrentPage（）

获取当前的页码。只在 slide模式 和 paging 启动下工作。

triggerRefreshOrLoad(type)

  • type ('refresh' | 'load')

直接触发刷新或者加载

Slot 下拉刷新/上拉加载动画

在 mode 为 slide 的情况下并且 pull-refresh / push-load 启用的情况下，可以给 Vuescroll 传入不同的 slot 来当作 pull-refresh / push-load 的不同阶段的提示。请确保 slot 的 dom 的 position 是 absolute，所有可能会引起一些 bug。

beforeDeactive 是加载完成后的提示阶段，但似乎有些问题。
slot 名称 对应显示阶段 描述
refresh-deactive deactive deactive 阶段显示的提示 dom
refresh-beforeDeactive beforeDeactive beforeDeactive 阶段显示的提示 dom
refresh-start start start 阶段显示的提示 dom
refresh-active active active 阶段显示的提示 dom
