Mint UI

Mint-UI，是真正的组件库，是使用 Vue 技术封装出来的成套的组件，可以无缝的和 VUE 项目进行集成开发。只能用在 vue 项目中，技术绑定。

官网：http://mint-ui.github.io/#!/zh-cn

安装 Mint UI

导入文件

完整导入

完整导入的包功能完全，但是体积较大。

按需导入

借助 babel-plugin-component 工具，我们可以只引入需要的组件，以达到减小项目体积的目的。

然后修改 .babelrc 文件。文件在项目根目录下，如果没有这个文件，自己创建，不过好像不创建也能使用的样子（2.x版本）。

使用 component 注册实现按需导入。

JS 组件

  • Toast：简短的消息提示框，支持自定义位置、持续时间和样式。
  • Indicator：加载提示框，支持自定义文本和加载图标。
  • Loadmore：下拉/上拉刷新，支持自定义 HTML 模板。
  • Infinite scroll：无限滚动指令。
  • Message box：弹出式提示框，有多种交互形式。
  • Action sheet操作表，从屏幕下方移入。
  • Popup：弹出框，可自定义内容。
  • Swipe：轮播图，可自定义轮播时间间隔、动画时长等。
  • Lazy load：图片懒加载指令。
  • Range：滑块，支持自定义步长、区间等。
  • Progress：进度条。
  • Picker：选择器，支持多 slot 联动。
  • Datetime picker：日期时间选择器，支持自定义类型。
  • Index List：索引列表，可由右侧索引导航快速定位到相应的内容。
  • Palette Button：调色板按钮。

使用 JS 组件需要在 .vue 文件的 script 中引入对应的包才行。

这里用 Toast 组件演示。

CSS 组件

css 组件有一下几种。

  • Header：顶部导航栏，支持显示按钮、自定义文字和固定在顶部。
  • Tabbar：底部选项卡，点击 tab 会切换显示的页面。依赖 tab-item 组件。
  • Navbar：顶部选项卡，与 Tabbar 类似，依赖 tab-item 组件。
  • Button：按钮，提供几种基础样式和尺寸，可自定义图标。
  • Cell：单元格，可用作展示列表信息、链接或者表单等。
  • Cell Swipe：可滑动的单元格，用法同 cell。
  • Spinner：加载动画，可指定显示类型、尺寸和颜色。
  • tab-container：面板，可切换显示子页面。
  • Search：搜索框，可显示搜索结果列表。

具体看文档，使用 Button 举例。对于 CSS 组件，需要在入口文件中按需导入即可。

Form 表单组件

  • Switch：开关。
  • Checklist：复选框列表，依赖 cell 组件。
  • Radio：单选框列表，依赖 cell 组件。
  • Field：表单编辑器。
  • Badge：徽章，可指定颜色和尺寸。

MUI 代码段

MUI 只是开发出来的一套好用的代码片段，里面提供了配套的样式、配套的 HTML 代码段，类似于 Bootstrap ，不涉及到技术捆绑，可以使用在任何项目。

官网：http://dev.dcloud.net.cn/mui/

由于 MUI 还不支持 npm 所以需要自己从 github 中下载资源插入到项目中。

解压，将 dist 中的三个编译好的资源放到我们自己的项目中就行，然后将需要的文件导入到入口文件技能正常使用了。

使用的话可以根据文档或者在下载资源的案例中复制代码就行，借鉴 bootstrap 用法 。

文档： http://dev.dcloud.net.cn/mui/ui/
