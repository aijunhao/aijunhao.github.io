Button 介绍

常用的操作按钮。

标签为 <el-button></el-button> ，可以定义按钮的尺寸（size），类型（type），朴素样式（plain），圆角（round），圆形（circle），加载（loading），禁用（disabled），图标（icon），是否聚焦（autofocus）等。

Button 属性

参数 说明 类型 可选值 默认值
size 尺寸 string medium / small / mini —
type 类型 string primary / success / warning / danger / info / text —
plain 是否朴素按钮 boolean — false
round 是否圆角按钮 boolean — false
circle 是否圆形按钮 boolean — false
loading 是否加载中状态 boolean — false
disabled 是否禁用状态 boolean — false
icon 图标类名 string — —
autofocus 是否默认聚焦 boolean — false
native-type 原生 type 属性 string button / submit / reset button

详情1：样式分析

:disabled

:disabled="buttonDisabled || loading" 主动设置 disabled 和 loading 属性值都会使按钮不可点击。

:class 绑定样式

type 类

type ? 'el-button--' + type : '' 拼接类名，只有 type 不为空是有类，不然不添加额外类，即使用默认样式。

注意，packages/theme-chalk/src/button.scss 按钮样式文件中有两种类型，一种是普通按钮，一种是按钮组。

详情2：vue 高阶组件之 provide 和 inject

官方 API 说明：https://cn.vuejs.org/v2/api/#provide-inject

文章参考：https://segmentfault.com/a/1190000014095107?utm_source=tag-newest

参数类型：

  • provide：Object | () => Object
  • inject：Array<string> | { [key: string]: string | Symbol | Object }
provide 和 inject 主要为高阶插件/组件库提供用例。并不推荐直接用于应用程序代码中。

简单的来说就是在父组件中通过 provide 来提供变量，然后在子组件中通过 inject 来注入变量。不管组件层次有多深，只要在上下游关系就能生效。相当于是强化了的 props。

此外，provide 和 inject 绑定并不是可响应的。
