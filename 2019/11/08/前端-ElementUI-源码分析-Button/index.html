Button 介绍

常用的操作按钮。

标签为 <el-button></el-button> ，可以定义按钮的尺寸（size），类型（type），朴素样式（plain），圆角（round），圆形（circle），加载（loading），禁用（disabled），图标（icon），是否聚焦（autofocus）等。

Button 属性

参数 说明 类型 可选值 默认值
size 尺寸 string medium / small / mini —
type 类型 string primary / success / warning / danger / info / text —
plain 是否朴素按钮 boolean — false
round 是否圆角按钮 boolean — false
circle 是否圆形按钮 boolean — false
loading 是否加载中状态 boolean — false
disabled 是否禁用状态 boolean — false
icon 图标类名 string — —
autofocus 是否默认聚焦 boolean — false
native-type 原生 type 属性 string button / submit / reset button

详情 1：样式分析

:disabled

:disabled="buttonDisabled || loading" 主动设置 disabled 和 loading 属性值都会使按钮不可点击。

:class 绑定样式

按代码所说，此处应该有三大类样式：

  1. el-button--primary ：颜色状态样式。
  2. el-button-buttonSize ：按钮大小样式。
  3. is-loading ：特殊状态样式。
并且，这三大类样式最终将处于父类 el-button 中，即 &。

在查看源码时需要特别注意，这里是将普通按钮和按钮组通用样式放在同一个文件中的，普通按钮是通过带参数的混入来实现的，而按钮组是通过直接比较直接的方式实现的，容易混淆，需要注意。

@include m()

m() 混入是用来生成 el-button--[primary/success/warning/danger/info/medium/samll/mini/text] 按钮颜色和大小的。

如上是 el-button--primary 样式的部分代码。在 m() 混入中 @each 遍历 $modifier （传入的参数），生成一个最终选择器 $currentSelector ：el-button--primary, xxx 。然后将 m() 中内容添加到根目录下。

button-variant() ，这个混入主要是设置该状态下的基本样式（字体色/背景色/边框色），鼠标悬浮和激活状态时的基本样式和 is- 状态下的基本样式。

详情 2：vue 高阶组件之 provide 和 inject

官方 API 说明：https://cn.vuejs.org/v2/api/#provide-inject

文章参考：https://segmentfault.com/a/1190000014095107?utm_source=tag-newest

参数类型：

  • provide：Object | () => Object
  • inject：Array<string> | { [key: string]: string | Symbol | Object }
provide 和 inject 主要为高阶插件/组件库提供用例。并不推荐直接用于应用程序代码中。

简单的来说就是在父组件中通过 provide 来提供变量，然后在子组件中通过 inject 来注入变量。不管组件层次有多深，只要在上下游关系就能生效。相当于是强化了的 props。

此外，provide 和 inject 绑定并不是可响应的。

详情 3：图标（拓展内容）

此样式是设置了图标的相邻兄弟 span 元素添加 5px 的左外边距。

有仔细的小伙伴应该注意到，组件中有两个图标占位，一个是 el-icon-loading 加载图标，一个是自定义图标，这两个只能显示一个或 0 个。并且都在文本左边！！！

如上为官网上的一个案例，我们可以很明确的看到，官网的做法是让使用者自己实现在右侧的情况，自己添加代码，传入到 slot 插槽中。从控制台样式上来看，el-icon--right 也有 5px 的左外边距，这样可以保证图标在左右不会影响到整体效果。

我们完全可以进行改造，添加一个字符串类型的 position 字段，来控制图标在左还是在右。
